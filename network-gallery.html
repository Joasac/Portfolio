Project Overview

This project involved designing a scalable and secure Local Area Network (LAN) infrastructure for differents settings. The design incorporates modern networking principles including network segmentation, automated IP allocation, and redundant routing to ensure reliable connectivity across all government departments.

Technologies Used

Cisco Packet Tracer Class C IP Addressing VLANs DHCP NAT RIP Protocol NITA-U Standards

Key Features

  • Departmental network segmentation using VLANs
  • Automated IP address allocation via DHCP
  • Internet connectivity through NAT configuration
  • Dynamic routing with RIP protocol
  • Redundant network paths for reliability
  • Security policies and access control

Challenges Overcome

Complex VLAN inter-communication setup, NAT configuration for multiple subnets, routing table optimization, and ensuring compliance with NITA-U networking standards for government institutions.

What I Learned

Advanced Cisco networking concepts, enterprise network design principles, routing protocol configuration, network security implementation, and documentation standards for government projects.
